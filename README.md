# 10day

10일 동안 머리속에 떠 오르는 생각들, 보고 들은 것들을 그냥 기록하고 싶다... 그러기 위해서는 독립왼 도메인하나를 가져야 겠다.

돈에 대하여 예전에 별 생각을 안해다. 하지만 요즘에 돈에 대하여 관리하기 시작했다. 그리고 어릴때 배워두었던 회계에 대하여 다시 생각하게 되었다.

“글이 살아 있는 것처럼” 움직이게 만드는 거
전체 글이었다가 → 자동으로 쪼개지고
다시 → 서로 합쳐지기도 하고
그리고 그 역사를 시각화하기
