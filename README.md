# 10day

10일 동안 머리속에 떠 오르는 생각들, 보고 들은 것들을 그냥 기록하고 싶다... 그러기 위해서는 독립된 도메인하나를 가져야 겠다.

돈에 대하여 예전에 별 생각을 안했다. 
하지만 요즘에 돈에 대하여 관리하기 시작했다. 
그리고 어릴때 배워두었던 회계에 대하여 다시 생각하게 되었다.

꽉찬 하루... 아침 7시부터 밤 11시 46분까지 가만히 있는 시간이 없는 하루... 하루 10분 생각할 시간이 필요하다. 동료들과 나눈 대화들 그리고 내 행동을 뒤돌아 본다.  도자기를 배우는데 역시나 쉽지 않다. 그리고 계속 실패하고 잘 안되면서 삽질하면서 배우는 것 같다. 고통은 늘 필요하다.
일은 어떻게 할건가... 그리고 어떤 도움이 될것인가를 늘 생각한다. 생각하면서 살아야 하겠다. 그리고 경청하고 남에 말을 들을 수  있어야 되겠다.
